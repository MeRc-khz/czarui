Interactive examples showing different tactical configurations. Deploy with precision.

settings Quick Configuration Reference
  • radius - Size of the dial circle (default: 120px)
  • justify - FAB position: left, center, or right (default: center)
  • slot="trigger-content" - Customize FAB button content
  • --primary - Override CSS variables for custom theming
