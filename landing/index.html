Full source code unlocks at the Team License ($149) and above. The Developer Edition also includes source code plus governance rights and lifetime dividends.

Can I use this commercially?

Absolutely. All licenses include commercial use rights.

Do you offer refunds?

Yes — 30-day money-back guarantee, no questions asked.

What browsers are supported?

All modern browsers: Chrome, Firefox, Safari, Edge. IE11 is not supported.

Do I need a framework?

No. Works with vanilla JS, React, Vue, Angular, or any framework that supports Web Components.

How do updates work?

You'll get email notifications when updates are released, with instant download access.

How does licensing work?

Each purchase generates a unique license key (BZRD-XXXX-XXXX-XXXX-XXXX) using HMAC-SHA256. Download tokens expire after 24 hours. Licenses never expire. Unlicensed usage shows a watermark overlay.

What's the IPFS config system?

An online configurator generates your custom config and uploads it to IPFS. The component fetches the config via CID, authorized by your API key. Fully decentralized — no centralized server dependency.

Is there a patent?

Yes — a provisional patent application was filed May 31, 2026 for the "Physics-Based Radial Dial Web Component with Integrated Inline Media Overlay System." Covers the hybrid Canvas/DOM rendering, inertial physics, and inline multi-modal content overlay.
