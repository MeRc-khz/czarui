# Technology Stack

## Overview
This project leverages modern web technologies to deliver a dynamic and interactive radial dial menu as a Web Component.

## Frontend
- **Languages:** JavaScript (ES6+), HTML, CSS
- **Frameworks/Libraries:**
    - **Web Components (Custom Elements):** For building reusable UI components.
    - **Leaflet.js:** An open-source JavaScript library for interactive maps (used for `data-map` functionality).
    - **Iconify:** For displaying vector icons (as seen in `icon` attributes).
- **Build Tools:**
    - **Vite:** A fast build tool that provides a lightning-fast development experience for modern web projects.

## Architecture
The project is structured as a Frontend Component Library / Single-Page Application (SPA), primarily focused on developing and demonstrating a single, highly interactive UI component.
